John's grunt file

What this does:

  • Compiles SASS and creates source maps
  • Minifies CSS
  • Compresses image files (PNG & JPG)
  • Syntax checks HTML and reports errors
  • Concatenates all JS files into single file named 'application.js'
  • Copies all source files to a 'build' folder ready for deployment
  • Fires up a HTTP server top serve pages
  • Watch process to auto-gen SASS files and livereloads all files upon save
